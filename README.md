# Data Structures Projects

A collection of four projects covering fundamental data structures and algorithms, developed for the **Data Structures** course at Ege University.

## Project 1 – Linked Lists & Core Structures

Two sub-projects implementing and utilizing linked list variations in C#.

- **Project1_1** – .NET 8.0
- **Project1_2** – .NET Framework 4.7.2

## Project 2 – Text Processing with Data Structures

Text analysis and processing using custom-built data structures in C# (.NET Framework 4.7.2).

## Project 3 – Advanced Data Structures

Implementation and benchmarking of advanced data structures in C# (.NET 9.0), with performance analysis.

## Project 4 – Trees, Graphs & Network Analysis

Multi-part project combining C# and Python:

| Question | Topic | Language |
|----------|-------|----------|
| Q2 | B-Tree Insertion | C# |
| Q3 | Breadth-First Traversal (BFS) | C# |
| Q3 | Dijkstra's Shortest Path | C# |
| Q3 | Prim's Minimum Spanning Tree | C# |
| Q4 | Social Network Analysis | Python |
| Q5 | Knowledge Graph | Python |
